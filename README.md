# Advent of code 2022

## Description

- **Language**: Node
- **Prompt**: [inquirer.js](https://github.com/SBoudrias/Inquirer.js/)

## Installation

```sh
yarn
```

## Usage

### Start a day

When you're ready to start a new day, run the command :

```sh
yarn new-day
```

### Solve a puzzle

You can test your script by running the following command :

```sh
yarn solve
```
